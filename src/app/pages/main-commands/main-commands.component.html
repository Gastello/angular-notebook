Angular CLI (Командний інтерфейс Angular)

Angular CLI – це інструмент для автоматизації створення компонентів, сервісів, модулів та запуску застосунку.

Основні команди

ng new project-name – створює новий Angular-проєкт.

ng serve – запускає локальний сервер для розробки.

ng generate component component-name або ng g c component-name – створює компонент.

ng generate module module-name або ng g m module-name – створює модуль.

ng generate service service-name або ng g s service-name – створює сервіс.

ng build --prod – збирає проєкт для продакшену.
